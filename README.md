Project Description
    This is a responsive Portfolio Service Website built using Next.js and Tailwind CSS to showcase developer skills, services, and contact information in a            professional layout.
    The website includes sections such as Hero, Services, Testimonials, Contact Form, and Footer.
    Vercel v0 was used to create the initial UI layout and design for multiple sections, which were later customized and integrated into the project structure.
    The project is optimized for performance, responsiveness, and clean code structure.
    The website is deployed on Vercel and accessible through a live link.
